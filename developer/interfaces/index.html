Interfaces and Utilities¶

pretalx provides a range of interfaces for developers, whether you want to add a new feature or develop a more complex plugin. Interfaces include models, model methods, and utility functions. The following pages also include reference material on how to handle internationalisation, permissions, email sending, long running tasks, and other nonstandard interfaces.
